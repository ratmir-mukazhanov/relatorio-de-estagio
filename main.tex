% ===================== RESUMO =================================================
\chapter*{Resumo}
\addcontentsline{toc}{chapter}{Resumo}

Este relatório descreve o trabalho desenvolvido durante o estágio curricular da Licenciatura em Tecnologias da Informação, realizado na empresa Devlop. O objetivo principal do projeto consistiu em evoluir uma API de seguimento de envios marítimos pré-existente (\textit{Tracking API}) através da integração com um novo fornecedor de dados: a plataforma SeaRates. O desenvolvimento de \textit{backend}, suportado por .NET e EF Core, envolveu a extensão do orquestrador de serviços e do registo de capacidades do sistema, de forma agnóstica e determinística. Foi desenhada e implementada uma interface \textit{frontend} em React para suportar a gestão operacional e administrativa da plataforma, incluindo operações \textit{CRUD} para múltiplas entidades, internacionalização e gestão de créditos comerciais.

Na fase final do estágio, embora o ambiente tenha sido preparado para a integração com o sistema interno de gestão de trânsitos (i-Forwarder) da empresa num ecossistema Four Js Genero (4GL), as prioridades do projeto foram reorientadas para garantir a estabilidade e a segurança da solução desenvolvida. O trabalho focou-se então na consolidação da integração com a SeaRates, na correção de anomalias complexas de dados geográficos no mapa de seguimento em tempo real e na migração integral do sistema de autenticação para o Keycloak. A utilização da solução em ambiente de produção permitiu observar o correto funcionamento da arquitetura implementada.

\vspace{1cm}
\noindent\textbf{Palavras-chave:} Integração de APIs, .NET, React, Seguimento Marítimo, Keycloak.

\newpage

% ===================== ABSTRACT ===============================================
\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This report describes the work conducted during the curricular internship for the Bachelor's Degree in Information Technologies at Devlop. The main objective of the project was to evolve a pre-existing maritime shipment-tracking API by integrating a new data provider, the SeaRates platform. The development involved backend components using .NET and EF Core, extending the system's service orchestrator and capabilities registry in an agnostic and deterministic manner. Additionally, a frontend interface was designed and implemented in React to support the platform's operational and administrative management, including CRUD operations for multiple entities, internationalization, and commercial credit management.

In the final phase of the internship, we prepared the environment for integration with the company's internal freight-forwarding system (i-Forwarder) within the Four Js Genero (4GL) ecosystem. However, priorities shifted to ensure the stability and security of the developed solution. The work focused on consolidating the SeaRates integration, fixing complex geographic data anomalies on the live tracking map, and fully migrating the authentication system to Keycloak. The results confirm the implemented architecture performs effectively under real production scenarios.

\vspace{1cm}
\noindent\textbf{Keywords:} API Integration, .NET, React, Maritime Tracking, Keycloak.
